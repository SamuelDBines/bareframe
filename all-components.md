# Bareframe Component Catalog

Large, reusable Web Components list for app development. Ordered alphabetically with short intent notes.

## A

- **accordion**: Expand/collapse grouped content panels.
- **action list**: Keyboard-friendly list of quick actions.
- **alert**: Inline status message for success, warning, or error.
- **anchor**: Styled link component with consistent states.
- **autocomplete**: Text input with dynamic suggestions.
- **avatar**: User profile image, initials fallback, and status.

## B

- **badge**: Small count or status label.
- **banner**: Full-width informational strip.
- **bar chart**: Compare values across categories.
- **bottom navigation**: Mobile-first tab-like app nav.
- **bottom sheet**: Slide-up panel for contextual actions.
- **breadcrumb**: Hierarchical path navigation.
- **button**: Standard clickable action trigger.
- **button group**: Related actions in one control cluster.

## C

- **calendar**: Month/week date layout and selection.
- **canvas**: Drawing surface for custom rendering.
- **card**: Content container with optional media/actions.
- **carousel**: Slide through items or media.
- **chart**: Generic chart wrapper with shared API.
- **checkbox**: Binary selection control.
- **chip**: Compact item/tag with remove action.
- **code block**: Syntax-highlighted code presentation.
- **color picker**: Select color with hue/saturation controls.
- **combobox**: Input + dropdown selection hybrid.
- **command palette**: Searchable command launcher.
- **context menu**: Right-click or trigger-based menu.
- **countdown**: Time-left display component.

## D

- **data grid**: Dense, sortable, filterable tabular data.
- **date picker**: Single-date input selector.
- **date range picker**: Start/end date selection.
- **dialog**: Modal or non-modal interaction window.
- **divider**: Visual separator between content groups.
- **donut chart**: Circular proportion chart.
- **drawer**: Side panel for navigation or tools.
- **dropdown**: Triggered list of choices.

## E

- **empty state**: Placeholder when no content exists.
- **error summary**: Form-level error collection and links.
- **expanded panel**: Toggleable region for advanced controls.

## F

- **fab**: Floating action button for primary actions.
- **file upload**: Pick/drag files with progress handling.
- **filter bar**: Active filter pills and controls.
- **footer**: Page/section footer wrapper.
- **form field**: Label/help/error wrapper around controls.

## G

- **gantt chart**: Timeline view of tasks and durations.
- **gauge**: Circular or linear progress meter.
- **graph**: Node/edge visualization container.
- **grid**: Reusable responsive layout primitive.

## H

- **header**: Page or section header container.
- **heatmap**: Intensity-based data visualization.
- **hero**: Prominent top-of-page marketing/info section.

## I

- **icon**: Inline icon renderer with size/color props.
- **image**: Lazy-load image with fallback handling.
- **input**: Base text-like input primitive.

## K

- **key-value list**: Display label/value pairs.

## L

- **line chart**: Trend over time visualization.
- **link**: Router-aware or standard hyperlink wrapper.
- **list**: Structured list container.
- **list item**: Standardized list row component.
- **loading indicator**: Spinner/bar for pending state.

## M

- **map**: Embeddable interactive map container.
- **menu**: Action/menu container with keyboard support.
- **menu item**: Selectable action inside menus.
- **message**: Chat/message bubble presentation.
- **metric card**: KPI display card.
- **modal**: Focus-trapped overlay dialog.
- **multi select**: Select multiple options with tags.

## N

- **nav**: Primary navigation wrapper.
- **notification**: Dismissible user notification item.
- **number field**: Numeric input with stepper behavior.

## O

- **otp input**: Multi-cell one-time-passcode entry.

## P

- **pagination**: Page navigation for long datasets.
- **pie chart**: Part-to-whole circular chart.
- **popover**: Anchored floating content panel.
- **progress bar**: Completion progress indicator.
- **progress circle**: Circular completion indicator.

## Q

- **quick actions**: Row of frequent action buttons.

## R

- **radio**: Single option within a radio group.
- **radio group**: Mutually exclusive option set.
- **range slider**: Min/max dual-thumb slider.
- **rating**: Star/score selector and display.

## S

- **search**: Search input with clear and submit behavior.
- **segmented control**: Compact single-select toggle set.
- **select**: Single-choice dropdown selector.
- **sheet**: Lightweight side/bottom panel.
- **skeleton**: Loading placeholder for content blocks.
- **slider**: Single-value draggable range input.
- **snackbar**: Temporary bottom message/toast.
- **sparkline**: Mini inline trend chart.
- **split button**: Primary action + secondary menu.
- **splitter**: Resizable two-panel divider.
- **stack**: Vertical/horizontal spacing layout helper.
- **stepper**: Multi-step progress/navigation control.
- **switch**: On/off toggle control.

## T

- **tab**: Individual tab trigger item.
- **tab group**: Tablist + panel management wrapper.
- **table**: Structured tabular data display.
- **tag**: Small label token for metadata.
- **textarea**: Multi-line text input control.
- **textfield**: Single-line text input control.
- **timeline**: Sequential events over time.
- **toast**: Ephemeral non-blocking message.
- **toggle**: Generic binary state control.
- **toolbar**: Row of grouped controls and actions.
- **tooltip**: Small anchored helper text bubble.
- **typography**: Semantic text styles for headings, body, labels, and code.
- **tree view**: Nested expandable hierarchy viewer.
- **treemap**: Hierarchical data in proportional rectangles.

## U

- **upload dropzone**: Drag-and-drop file receiver.

## V

- **video player**: Media playback controls wrapper.
- **virtual list**: Windowed rendering for large lists.

## W

- **wizard**: Guided step-by-step flow container.
